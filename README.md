\*예외처리(2가지)

1. 컴파일 에러
   =>컴퓨터번역에러
   =>프로그램 실행되기 전에 발생하는 오류
2. 런타임 에러
   =>프로그램이 실행 중 발생하는 오류

\*try ~catch(사용)
=>예외가 발생할 거 같은 코드를 제어함
try{
예외 발생 가능성이 있는 코드작성(문제발생)
}catch(변수명(error)){
예외가 발생했을 때 처리
}finally{
예외와 상관없이 무조건 실행해야하는 코드
}
